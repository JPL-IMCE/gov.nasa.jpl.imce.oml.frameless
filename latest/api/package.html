package root

Frameless-based SPARK API for the Ontological Modeling Language

1. OML Frameless API

Each concrete OML metaclass has a corresponding OML Frameless API case class.

oml.frameless.api

Note that all UUID fields are typed with a covariant tag. This provides compile-time verification for UUID cross-references.

2. Frameless-based Apache Spark Dataset for OML

This case class is a big tupe of Frameless-based Apache Spark TypedDatasets, one for each of OML's concrete metaclasses in the gov.nasa.jpl.imce.oml.frameless.api.

oml.frameless.OMLSpecificationTypedDatasets

3. Apache Parquet file conversions for OML

oml.frameless.conversions.Parquet2Tables

This command-line tool takes two arguments: - A path to a directory with *.parquet files that will be read - A path to a *.omlzip file which will be written with the OML contents read from the parquet files.

oml.frameless.conversions.Tables2Parquet

This command-line tool takes two arguments: - A path to an OASIS XML Catalog file named oml.catalog.xml from which all *.omlzip files will be read - A path to directory which will be written with parquet files for OML
